⚠️ Important Notice

THIS IS A DEMO SITE NOT MEANT FOR REAL USE. ANY DATA YOU SHARE WILL BE SEEN BY THE DEVELOPERS.

This is an open source experimental prototype. It is not intended to provide official advice or replace professional assistance. Do not rely on these tools for emergency response. Always contact official emergency services and follow local authority guidance.

Data Privacy: We do not store or retain any user data or conversations. Each session is temporary and data is not persisted. You can verify this by reviewing our open source code which shows that no database or storage systems are used for user data.
